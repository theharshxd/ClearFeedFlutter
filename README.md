# ClearFeed

Silently blocks doom scrolling on social media apps.

## What it blocks
- YouTube Shorts
- Instagram Reels
- Facebook Reels
- TikTok full feed
- Snapchat Spotlight
- X video feed
- Reddit video feed

## What it does NOT block
- Normal feed reading
- Comments
- Chat / DMs
- Watching single videos
- Normal browsing

## How it works
Uses Android Accessibility Service to detect when you open video feed screens and silently returns you to the main page. Zero battery drain when target apps are closed.

## Install
1. Download APK from GitHub Actions
2. Install on phone (allow unknown sources)
3. Open ClearFeed
4. Enable Accessibility Service
5. Done
